# Final Image Assets

Put manually edited/downloaded production images here.

Recommended workflow:

1. Use the source files from `VIsuals/images`.
2. Apply the prompt from `docs/site-redesign-audit/05-image-edit-prompts-and-placement.md`.
3. Save the edited file with the exact target filename listed in that prompt sheet.
4. Prefer `.webp` for photos and banners, `.png` for transparent logos.

Folders:

- `brand`: logo and brand marks.
- `hero`: first-viewport hero banners.
- `people`: personal/portrait images of Shahriar Vision leadership.
- `services`: service page visuals.
- `articles`: blog/article thumbnails.
- `backgrounds`: footer/contact/section backgrounds.
- `incoming-manual-edits`: temporary drop zone if a final destination is not decided yet.

Do not reference files from `incoming-manual-edits` in production code.
